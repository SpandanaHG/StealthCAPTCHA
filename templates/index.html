Research Objectives

Investigating behavioral biometrics for enhanced cybersecurity

Behavioral Pattern Analysis

Investigate mouse movement dynamics, keystroke patterns, and interaction timings to identify unique behavioral signatures distinguishing human users from automated bots.

Machine Learning Classification

Develop and evaluate Random Forest algorithms for real-time classification of user behavior, achieving high accuracy in distinguishing legitimate users from malicious bots.

Cybersecurity Enhancement

Propose a novel approach to web security that eliminates user friction while maintaining robust protection against automated attacks and malicious activities.

Biometric Feature Extraction

Research optimal feature sets from behavioral data including velocity patterns, rhythm consistency, and spatial distribution metrics for accurate user identification.

Real-time Processing

Develop efficient algorithms capable of processing behavioral data and making classification decisions within milliseconds for practical web application deployment.

Performance Evaluation

Comprehensive evaluation of model performance using precision, recall, F1-score, and ROC curves to validate the effectiveness of behavioral biometric classification.
